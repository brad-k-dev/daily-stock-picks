  Brad's Daily Stock Picks — GitHub Pages 배포 가이드
════════════════════════════════════════════════════════════

📁 파일 구성
  index.html              → 주식 추천 웹페이지 (코드 잠금 포함)
  deploy.sh               → 최초 배포 스크립트 (한 번만 실행)
  update_picks.py         → AI 자동 갱신 Python 스크립트
  .github/workflows/
    daily-update.yml      → 매일 오전 9시 자동 실행 워크플로우

────────────────────────────────────────────────────────────
🚀 최초 배포 방법 (Mac 터미널에서)
────────────────────────────────────────────────────────────

1. 이 폴더를 원하는 위치로 이동
   예) ~/Documents/daily-stock-picks/

2. 터미널 열기 (Finder → 폴더 우클릭 → 터미널에서 열기)

3. 스크립트 실행:
   bash deploy.sh

4. GitHub 사용자명과 PAT 토큰 입력 시 프롬프트에 입력

5. 완료 후 표시되는 URL 확인:
   https://[사용자명].github.io/daily-stock-picks/

6. GitHub Pages 빌드 완료까지 1~2분 대기

────────────────────────────────────────────────────────────
🔐 접근 코드 변경 방법
────────────────────────────────────────────────────────────

index.html 파일을 텍스트 에디터로 열고 아래 줄을 찾아 수정:

  const ACCESS_CODE = 'BRAD2026';  ← 이 값을 원하는 코드로 변경

변경 후 GitHub에 다시 푸시:
  git add index.html
  git commit -m "코드 변경"
  git push

────────────────────────────────────────────────────────────
🤖 AI 자동 갱신 설정 (선택사항)
────────────────────────────────────────────────────────────

매일 오전 9시 AI가 최신 추천을 자동 갱신하려면:

1. Anthropic API Key 발급:
   https://console.anthropic.com/settings/keys

2. GitHub 저장소 Secrets에 등록:
   github.com/[사용자명]/daily-stock-picks/settings/secrets/actions
   → "New repository secret" 클릭
   → Name: ANTHROPIC_API_KEY
   → Secret: sk-ant-... (발급받은 키)

3. 이후 매일 오전 9시 자동 실행됨
   (Actions 탭에서 수동 실행도 가능: "Run workflow" 버튼)
